AT&T API Platform Adapter for IBM® Worklight®


The AT&T API Platform adapter for IBM Worklight provides a simplified way for Worklight developers to access the AT&T API Platform and RESTful APIs. By significantly reducing the complexity of building applications that use the AT&T API Platform, the adapter helps developers quickly bring robust hybrid mobile applications to market.


Worklight is an advanced, comprehensive mobile development and server platform for smartphone and tablet apps. Using familiar open technologies, organizations can efficiently develop, connect, run, and manage mobile-web, native, and hybrid application types. It can help reduce development time, cost, and complexity, and enable a better user experiences on a variety of mobile devices.


Worklight lets developers build rich, cross-platform apps without needing code translation, proprietary interpreters, or specialized scripting languages. The platform supports standards-based technologies and tools, including HTML5, CSS3, JavaScript, Apache Cordova, and popular JavaScript frameworks such as Dojo Mobile, jQuery, and Sencha Touch.


Worklight ships with a full-featured development environment, mobile-optimized middleware, and an integrated management and analytics console, supported by a variety of security mechanisms.
